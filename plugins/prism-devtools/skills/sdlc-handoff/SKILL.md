Creates a complete handoff package (SDLC doc, task assignments, comms plan) for Dev/QA from a validated customer issue.

## Steps

1. Compile issue package: validation evidence, investigation, impact
2. Create SDLC workflow document with executive summary
3. Generate task assignments for Dev, QA, and DevOps teams
4. Execute communication plan (internal + customer update)
5. Establish tracking metrics and escalation rules

See [instructions.md](./reference/instructions.md) for document templates and task assignment matrix.
